4) Why isn’t it possible to see the first five rows of a table?
Ans : In RDBMS rows are store in table. table is not a file internally every row is file. when we insert row in table whenever it finds free space in the DB server HD it will store there. so row is not store in sequential manner so its not possible to see first five rows of the table.
